### Task 7 -- Create /src/scenes/GameScene.js

This is the main Sprint 1 deliverable. In `create()`:

1. Create the tilemap: `const map = this.make.tilemap({ key: 'ahupuaa-map' })`
2. Add the tileset: `const tileset = map.addTilesetImage('terrain', 'terrain')`
3. Create the terrain layer: `const layer = map.createLayer('terrain', tileset, 0, 0)`
4. Call `buildWorld()` to initialize `state.tiles[col][row]`
5. Iterate all cols (0-19) and rows (0-55). For each tile:
   - Get pixel center: `const { x, y } = map.tileToWorldXY(col, row)`
   - Determine atlas frame from terrain type (see CLAUDE.md terrain frame table)
   - Place a Phaser Image: `this.add.image(x, y, 'terrain', frameName)`
   - Apply tint if needed (ocean/stream: 0x2255bb, forest: 0x1a5c1a)
6. Log pixel centers for 3 specific test tiles to console:
   `console.log('tileToWorldXY test:', map.tileToWorldXY(0, 0), map.tileToWorldXY(10, 28), map.tileToWorldXY(19, 55))`
7. Log spot-check tile data:
   `console.log('tiles spot check:', state.tiles[0][0].terrainType, state.tiles[10][28].terrainType, state.tiles[19][55].terrainType)`

### Task 8 -- Rewrite main.js

Replace prototype wiring with Phaser game config and scene launch.

```js
import Phaser from 'phaser';
import PreloadScene from './src/scenes/PreloadScene.js';
import GameScene from './src/scenes/GameScene.js';

const config = {
  type: Phaser.AUTO,
  width: 1200,
  height: 900,
  backgroundColor: '#1a1a2e',
  roundPixels: true,   // Phaser 4: default flipped from true to false; set explicitly. See PHASER_4_AUDIT.md.

};

new Phaser.Game(config);
```

## Atlas Frame Reference

Apply this mapping in GameScene when choosing which frame to render per tile.
Use `map.tileToWorldXY` for positioning -- never compute pixel centers manually.

| terrainType | Atlas frame | Tint |
|---|---|---|
| `flat` | `grass_01.png` | none |
| `mountain` | `stone_01.png` | none |
| `peak-tip` | `stone_01.png` | none |
| `peak-upper` | `stone_01.png` | none |
| `shore` | `sand_01.png` | none |
| `ocean` | `grass_01.png` | 0x2255bb |
| `stream` | `grass_01.png` | 0x2255bb |
| `forest` | `grass_01.png` | 0x1a5c1a |

## Architectural Constraints (enforced this session)

- `/src/game/` files must have ZERO Phaser imports. If a game file imports Phaser, it is in the wrong directory.
- `tiles[col][row]` is column-first. Do not invert.
- `map.tileToWorldXY(col, row)` is the only correct Tiled-to-pixel function. Do not compute manually.
- Do not modify `getNeighbors()` or any neighbor array in HexGrid.js.
- Do not start building placement, resource tick, or river system. Those are Sprint 2.
